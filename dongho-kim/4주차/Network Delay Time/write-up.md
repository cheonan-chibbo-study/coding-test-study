# 문제 링크
https://leetcode.com/problems/network-delay-time/description/

# 접근 방법
문제에서 그래프가 주어질 때, k 노드에서 시작해서 다른 모든 N개의 노드까지 모두 방문하는 데 걸리는 최소 시간을 찾는 문제이다.

이때 문제 제약 조건 중, 간선의 크기가 음의 정수가 아니므로 **다익스트라** 알고리즘 적용이 가능하다.

따라서 우선 주어진 데이터로 연결 그래프를 만들고, 다익스트라 알고리즘을 k 노드부터 시작하여 수행한다.

그 다음, 각 노드에 기록된 최단 거리를 보면서 정답을 출력한다.
