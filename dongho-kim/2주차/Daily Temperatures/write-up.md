# 문제 링크
https://leetcode.com/problems/daily-temperatures/description/

# 접근 방법
1. (완탐) 2중 for문을 돌린다. -> 10^10번의 연산으로 시간초과
2. 온도를 key로, 인덱스 리스트를 value로 저장 -> 각 온도 순회하면서 그 온도보다 높은 온도를 모두 탐색 -> 최악의 경우 10^10
3. 스택에 아직 더 따뜻한 날을 찾지 못한 인덱스를 저장 -> 각 온도 순회하면서, 스택에 저장한 인덱스의 값이랑 비교해서 차이 기록

# 배운 점
1. 사실 아직 이게 왜 스택을 써야하는 지 이해가 잘 가지 않는다. 내일 다시 봐야겠다.
