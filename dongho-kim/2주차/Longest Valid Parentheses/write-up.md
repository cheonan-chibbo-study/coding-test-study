# 접근 방법
처음엔 단순히 Stack 활용해서 '(' 가 나오면 삽입하고, ')' 가 나오면 `pop()` 하면서 길이를 카운팅하는 방식으로 구현했습니다.  
하지만 반례가 존재했고, 그 반례를 해결하면 다른 테스트 케이스를 만족하지 못했습니다.

그래서 문자열의 뒤에서부터 접근해보는 방식도 생각해봤습니다. 하지만 앞에서 가는 것과 다를 게 없었습니다.

또 다른 방법을 생각하다가, '('와 ')'이 매치된 인덱스 각각에 마킹을 해두고(`matched[i] = True`), 마킹 배열을 순회하면서 최장 길이를 구하는 식으로 구현했습니다.

문자열의 길이를 N이라고 했을 때, 시도한 모든 방법의 시간 복잡도는 O(N) 입니다. 

# 배운 점
1. 스택 하나로 한번에 해결할 생각을 버리고, 또 다른 자료구조를 이용해서 푸니까 쉽게 풀린다. 그런데 이를 생각하기가 시간이 걸린다ㅜㅜ. 그래도 다음에 비슷한 문제가 나오면 쉽게 떠오를 수 있을 것 같다.
