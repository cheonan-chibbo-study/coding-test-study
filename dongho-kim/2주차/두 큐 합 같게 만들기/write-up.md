# 접근 방법
q1과 q2의 총합을 비교해서, 총합이 더 큰 큐의 요소를 하나 빼서 반대편 큐에 삽입하는 로직을 반복하는 식으로 접근했다.

그런데 처음에는 큐의 총합을 매 while 루프마다 계산을 해서 시간초과가 발생했다. 파이썬의 `sum()` 함수는 리스트를 모두 순회하므로 O(N)의 시간이 소요되기 떄문이다.  

그래서 sum1이랑 sum2를 while문 밖으로 뺐다. 그럼에도 시간초과가 발생했다.

큐에 요소를 추출하고 삽입하는 과정이 무한정 반복되는 것 같아서 최대 반복 횟수를 정했다. 큐의 길이의 3배만큼 반복하면 반드시 한 번은 원점으로 돌아오기 때문에 3배로 잡았다.
