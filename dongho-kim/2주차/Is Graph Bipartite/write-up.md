# 문제 링크
https://leetcode.com/problems/is-graph-bipartite/description/

# 접근 방법
이분 그래프 개념을 적용하면 되는 문제였다.  

이분 그래프는 모든 정점을 빨간색, 혹은 파란색으로 칠했을 때, 모든 간선에 대해서 빨간색이랑 파란색을 포함하도록 하는 그래프를 뜻한다.

따라서 DFS를 활용해서 특정 노드에 연결된 모든 노드의 색상을 비교해서, 아직 방문한 적이 없다면 현재 노드와 다른 색상을 칠한다. 반대로 연결된 노드에 방문한 적이 있는데 칠해진 색상이 현재 노드와 같은 색상이면 이분 그래프가 될 수 없다.  

위를 재귀 함수로 구현했다.

# 배운 점
1. 이분 그래프에 대해 상기했다. 그리고 직접 이분 그래프 구현 방식을 고민하면서 코드를 짠 건 처음이라 좀 더 연습이 필요하다.
