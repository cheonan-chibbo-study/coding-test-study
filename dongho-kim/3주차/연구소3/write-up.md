# 문제 링크
https://www.acmicpc.net/problem/17142

# 접근 방법
N이 최대 50, M은 최대 10인데 Python3 기준 시간 제한이 1.5초라서 자신있게 완전 탐색으로 접근했습니다.

문제에서 요구하는 대로 순서대로 구현했습니다.

1. 입력으로 주어진 바이러스의 좌표를 리스트로 저장합니다.
2. 비활성 바이러스 중에서 M개를 골라서 활성 상태로 만듭니다. -> 재귀 함수
3. 시뮬레이션 돌려서 바이러스를 퍼트린다. -> 빡구현
4. 이후에 결과를 구합니다.

접근 순서는 맞았지만, 문제에서 요구하는 바를 정확히 파악하지 못해 자꾸 틀렸습니다.
문제에서 요구하는 바는 **빈 칸**에 퍼뜨리는 최소 시간이었는데, 비활성화 상태의 바이러스에 퍼진 시간까지 포함하다보니 틀렸습니다.

문제가 길기도 하고 테스트케이스 만들기 귀찮아서 생략했더니 혼쭐이 났네요..

# 배운 점
1. 문제를 정확히 이해하지 않고 테케 안만들거면 여러 번 제출할 각오하자!
