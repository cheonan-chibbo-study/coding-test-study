# 문제 링크
https://leetcode.com/problems/shortest-path-in-binary-matrix/

# 접근 방법
처음에는 DFS로 구현할 수 있지 않을까 싶어서 DFS로 구현했습니다. (약 10분 소요)
구현하고 나서, 사실 문제에서 요구하는 건 (0,0)부터 (n-1, n-1)까지의 **최단 경로를 요구**하는 문제임을 다시 깨달았습니다.

최단 경로는 BFS가 더 적합한 알고리즘이기 때문에 BFS로 다시 구현했습니다. (약 10분 소요)

# 배운 점
- 그래프 상의 최단 경로를 구할 땐 BFS다.
