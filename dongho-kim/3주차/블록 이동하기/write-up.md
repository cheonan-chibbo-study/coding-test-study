# 접근 방법
이 문제의 태그에 BFS인 것을 의식하지 않고, 처음엔 시뮬레이션으로 접근하면 될 것 같았습니다.  
하지만 큐에 이동할 수 있는 경로를 넣어두고 모두 탐색하는 것이 낫겠다는 생각이 들어 자연스레 큐를 사용하게 되었고 BFS 형태로 구현하게 됐습니다.

이후 좌우/상하 이동, 8가지의 회전 경우를 모두 구현해서 방문하지 않은 경우이면 모두 탐색하도록 구현하려고 했으나, 방문 처리에 대한 구현 아이디어를 떠올리지 못했습니다.  
그리고 회전하는 8가지 경우의 수를 구현하려고 했으나, 효율적인 방법을 찾지 못해 시간 안에 못풀 것 같아 AI에게 도움을 요청했습니다.

나중에 혼자서 다시 풀어봐야겠습니다.

# 배운 점
1. 각 좌표를 튜플로 묶고, 두 좌표를 하나의 집합으로 묶어서 방문 처리 할 수 있다.
