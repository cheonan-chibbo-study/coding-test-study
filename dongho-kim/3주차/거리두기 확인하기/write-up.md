# 접근 방법
각 place를 (0, 0)에서 (4, 4)까지 순회하면서 `place[x][y] == "P'` 라면, 해당 지점부터 격자 탐색을 시작합니다.

이때 `place[x][y]`가 `"O"`로 이어지면서 동시에 맨해튼 거리가 2 이내에 또 다른 P가 있는지만 판별하면 된다고 생각했습니다.

그래프에서 정점 간의 거리를 구할 때에는 BFS 알고리즘이 적합하므로 BFS 방식으로 탐색을 진행했습니다.
