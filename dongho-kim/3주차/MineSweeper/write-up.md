# 문제 링크
https://leetcode.com/problems/minesweeper/description/

# 접근 방법
문제를 잘 살펴보면 요구하는 바는 단순합니다.  
지뢰찾기 게임판이 주어지고 한 칸을 클릭했을 때의 결과판을 반환하면 됩니다.

만약 클릭한 지점이 지뢰가 있다면 해당 칸만 "X"로 바꿔줍니다.  
그게 아니라면 클릭한 칸의 주위 여덟 방향을 탐색하면서 지뢰의 개수를 셉니다. 만약 1개라도 있다면 해당 칸을 카운트 한 개수로 바꿔주고, 0개라면 클릭한 칸을 "B"로 바꿔주고 주위 모든 칸을 계속해서 탐색합니다.

BFS를 이용하여 쉽게 풀었습니다.

# 배운 점
1. 직관적이고 단순한 BFS 문제여서 쉽게 풀었다. 오히려 이렇게 쉽고 직관적인 문제를 푸니까 BFS를 머리에 그리기 좋은 문제였다.
