# 문제 링크
https://leetcode.com/problems/sudoku-solver/description/

# 접근 방법
모든 칸을 탐색하면서 `'.'` 칸에 모든 숫자를 넣어본다. 이때 스도쿠 게임 조건에 따라 배치 가능한 숫자를 배치하고 다음 칸으로 넘어가는 로직을 반복한다.

숫자를 배치할 수 있는지 체크하는 알고리즘을, 처음엔 `(x, y)` 기준으로 행, 열, 3x3 공간을 직접 하나씩 탐색했다.  
그런데 시간 초과가 발생해서 Set을 이용하는 방식으로 수정했다.

# 배운 점
1. `if self.recurse(...): return True` 패턴은 백트래킹에서 해를 찾은 즉시 모든 탐색을 중단시키는 가장 표준적인 방법이다.
