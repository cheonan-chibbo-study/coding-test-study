# 문제 링크
https://leetcode.com/problems/n-queens/

# 접근 방법
모든 칸에 대해서 퀸을 배치하는 경우를 따져봅니다.

이때, 퀸의 경우 x행에 배치하면 동일한 행에는 더이상 배치할 수 없기 때문에 바로 다음 행에서 탐색을 시작합니다.  
그리고 배치하기 전에 배치가 가능한지 판별하여, 가능한 경우에만 배치하도록 구현했습니다.  

배치가 가능한지 확인하는 연산은 여덟 방향을 모두 탐색하며 퀸이 있는지를 확인했는데요.  
만약 연산을 최적화하려면 퀸을 배치한 좌표를 따로 리스트로 저장한 다음, 각 리스트를 돌면서 배치하려는 (x, y) 좌표와 수직 또는 수평을 이루는지? 혹은 `|x - x'| == 1 and |y - y'| == 1` 인지 계산하면 더 최적화할 수 있겠지만, 그러지 않아도 해결이 가능해서 패스했습니다.
