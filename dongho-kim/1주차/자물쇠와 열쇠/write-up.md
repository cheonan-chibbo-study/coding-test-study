# 접근 방법
이동과 회전이 된다는 문장을 보고, 이동도 해보고 회전도 다 해보자는 생각으로 접근했습니다.

구현 문제다보니 초반에는 문제 요구사항 정리하고, 메서드 단위로 설계를 해 나갔습니다.

그 다음부터는 열심히 구현했습니다.

# 배운 점
1. 빡구현 문제는 역시 설계를 잘 해야 하고, 출력 찍어가면서 하나씩 완성해가는 게 빠르다.
2. 빡구현 문제 풀 때 일부 조건을 놓치는 경우가 종종 있다. 실전에서는 사소한 실수가 시간을 너무 잡아먹는데 이를 어떻게 개선해야 할까?
3. `key_x`랑 `key_y`를 구하는 부분에서 시간을 좀 썼는데, 감으로 때려 맞혔다.. 배웠다기보다 이거 이해를 좀 해야겠다.
4. 배열 90도 회전하는 코드를 공부했었는데 기억이 안났다. 중간에 살짝 보고 풀어버렸다. 아직 원리가 이해가 안되서 그런 것 같은데 원리도 이해하고 코드도 외워야겠다.
