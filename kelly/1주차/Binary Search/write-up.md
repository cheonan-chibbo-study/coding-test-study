## 문제 분석
주어진 nums 배열에서 값이 target인 요소의 인덱스를 찾아 반환하는 문제이다.

- 대상 값이 존재하지 않으면 -1 반환

문제에서 주어진 조건은 다음과 같다.

- `1 <= nums.length <= 104`
- `104 < nums[i], target < 104`
- All the integers in `nums` are **unique**.
- `nums` is sorted in ascending order.

### 이진 탐색

nums 배열이 오름차순으로 정렬되어 있고, 문제에서 무조건 `O(log n)` 의 시간 복잡도로 풀어야 한다고 강조하는점을 보아 이 문제는 이진 탐색을 연습하도록 하는게 목적인 문제임을 알 수 있다.

따라서 이 문제는 이진 탐색을 활용하여 정답 코드를 작성한다.

### 결론

- 이 문제는 이진 탐색으로 해결해야한다.
