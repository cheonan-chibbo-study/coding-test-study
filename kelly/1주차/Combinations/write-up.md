## 문제 분석
1 ~ n 범위의 정수 중 k개의 조합을 모두 생성해 반환하는 문제이다.

주어진 문제 입력 값 범위는 다음과 같다.

- `1 <= n <= 20`
- `1 <= k <= n`

참고로 n개의 정수 중 k개의 조합을 모두 구하는 공식은 다음과 같다.

- `$nCk$` == `$n! / k!(n - k)!$`

이 때 n, k의 최대값은 20이므로 최대로 구할 수 있는 조합의 개수는 다음과 같다.

- n이 20이고 k가 10(n / 2)일 때
- 20C10 == `184,756개`

### 완전 탐색 (백트래킹)
조합 하나의 길이가 k이므로 조합 하나를 구하는데 k번의 연산이 필요하다고 가정하면 최악의 경우 시간 복잡도는 다음과 같다.

- n = 20, k = 10
- 생성할 수 있는 조합의 총 개수 → `184,756개`
- 최종 시간 복잡도는 O(10 * 184,756) == `O(1,847,560)`

결론적으로 최악의 경우 약 180만정도의 연산이 필요하므로 백트래킹을 활용해 충분히 이 문제를 풀 수 있다.

### 결론
- 백트래킹을 사용해 문제를 해결할 수 있다.
