## 👩‍🚀 문제 분석

주어진 스토쿠 문제를 해결한 답을 문자열로 반환하는 문제입니다.

문제에 주어진 제약 조건은 다음과 같습니다.

**Constraints:**

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` is a digit or `'.'`.
- It is **guaranteed** that the input board has only one solution.

## 🤔 풀이 고민

### 완전 탐색 (백트래킹)

이미 격자의 크기가 9라는 작은 숫자로 고정된 입력이 들어오는 문제이기 때문에 백트래킹 연습 문제라는 촉이 왔습니다.

전체 격자를 순회하면서 빈 격자가 나오면 1 ~ 9의 숫자중 현 상황에서 대입할 수 있는 정수를 삽입하고, 만약 삽입할 수 있는 정수가 존재하지 않는다면 해당 케이스를 포기하는 방식으로 백트래킹 풀이를 대입하면 됩니다.
