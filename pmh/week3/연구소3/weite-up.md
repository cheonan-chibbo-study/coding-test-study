# 접근방법

1. 보드에서 2 인 모든 바이러스를 미리 저장하고 빈칸인 0번 을 미리 다 저장한다.
2. 현재 들어온 비활성화 바이렁스중 M 개를 선택해 바이러스를 활성화 하므로 재귀 방식을 써서 활성화 가능한 모든 경우의 수를 구한다
3. M 개까지 바이러스를 활성화시 BFS 를써서 바이러스를 전파시키고 보드가 0 인경우를 전파시켰을때만 초를 센다
4. 미리 빈칸을 저장해둔 갯수와 바이러스 전파 개숫를 비교해 같다면 모든 보드를 전파 시킨것으록 간주해 최소 시간을 갱신한다
