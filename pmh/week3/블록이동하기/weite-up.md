# 접근방법
1. 최단 경로를 탐색하는거라 BFS 를 사용하면될것같다고 생각했다
2. 일단 상하좌우 이동시 벽을 만나면  회전 하는 방향으로 풀면 되겠다고생각했다.
3. 또한 회전할시 회전할 방향의 바로 아래 쪽에 벽이있다면 넘어가고 다음 회전을 하는 방식으로 하면 될것같다고 생각했다.
4. 그러나 로봇이 2칸을 차지한다는 점 떄문에 구현이 어떻게 해야할지 떠오르지 않아서 30분을 초과했다
5. 결국 구현하지 못해서 검색후 공부하면서 풀게됬다.

- 너무 어렵다 다른 사람 문제풀이봐도 다들 기본 100 줄이넘어갔다 먼 이런 문제가 다있는지...
- 로봇 처럼 여러 정보를 가진 거를 다룰때는 클래스를 쓰는것도 좋은생각이라는것을 배웠다.
