# 문제링크

https://leetcode.com/problems/coin-change/

# 접근방법
1. 문제 유형을보고 bfs 써야 겠다고 생각하긴 했음
2. BFS 를 구현할줄 몰라서 BFS 에대해 찾아보고 풀어 볼려고 했음
3. 그러나 30분동안 어떻게 풀지를 생각 해내지 못해서 결국 검색해서 해결함
4. BFS 를 써서 0원 에서 출발해서, 동전들을 계속 더해가면서 더한 동전 갯수를 세고 총합과 맞으면 동전 갯수를 반환하는 문제로 생각했다.
    - [1,2,5] , 12 라고 가정할때
    - 처음에 큐에 0 을 넣는다 
    - 코인 배열에서 코인을 꺼내와 큐에 1,2,5 를 넣는다
    - 그후 코인갯수를 나타나는 steps 에 1을 더한다. steps= 1
    - 이제 큐는 다시 1 를 꺼내온다
    - 꺼내온 1을 [1,2,5] 에 다 더해서 [2,3,6] 을 만들고 visited 를 써서 이 숫자가 나온적이없으면 큐에 수를 넣는다.
    - 그후 2 꺼내서 [3,5,7] 를 만들고 나온적 없는 숫자면 큐에 넣기
    - 5꺼내서 [6,7,10] 를 만들고 나온적 없는 수자면 큐에 넣기
    - 다 더하면 steps 에 1를 더한다  steps =2
    - 이러면 큐에 [2,3,6,5,7,10] 생긴다 -> 코인 2개 를 이용해서 만들수있는 모든 수 가 생김 (중복 x)
    - 큐에 12인 수가 들어오면 현재 steps를 반환한다.
