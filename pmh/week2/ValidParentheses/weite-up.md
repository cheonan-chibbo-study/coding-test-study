# 접근방법
 1. stack 사용해야하는 거는 떠올렸지만 푸는방법 이 안떠올라 leet 코드 문제에있는 힌트를 봤다
2. 만약 여는 괄호 가 나오면 스택에 푸쉬하고 닫는 괄호가 나오면 스택이 비어있는지 or 해당 괄호와 짝 되어있는 여는 괄호인지 확인후 스택이 비어있거나 짝 괄호가 아니라면 false 를 반환한다
3. 또한 map 를 써서 불필요한 if문의 갯수를 줄였다
4. 또한 마지막에 for문 무사히 끝내고 바로 true 를 하는게아닌 stack.isEmpty() 로 리턴한다 그이유는 괄호가 '(' 하나만 들어오면 for문을 정상적으로 통과해버리기 때문에
마지막에 스택에 완전히 비어있는지 확인하는걸로 return 을 보내 괄호가 하나인경우에도 대비한다.
