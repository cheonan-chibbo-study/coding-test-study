# 접근방법

n 개의 방이 있고 방들이 0~n-1 호 라벨이 되어있음
모든 방들음 0 호 룸들으 제외하고 다 잠겨있음
모든 방을 방문하는게 목표
그러나 키없이 잠겨진 방을 들어갈수는 없다
너가 방을 방문할때 방안에서distinct keys 들 을 찾을수있음
각각 키들은 문의 번호를 가지고있고



1. 0번 방들어가서 키 가져오기
2. 가져온 키로 키의 번호에 맞는 방문 열기
3. 이미 간적 있는 방문이면 넘어가고
4. 방문한적 없는 방이면 방문열고 키들을 스택에 담기
5. 방문한 방의 갯수를 +1 하기
6. 스택이 비어있다면(키로 열수있는 방문을 다열었다면)
7. 마지막 방문한 방의 갯수와 rooms 의 사이즈가 같으면 true 다르면 false 반환
