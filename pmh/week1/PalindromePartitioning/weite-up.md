# 접근방법
palindrome(팰린드롬) : 잘린 조각들이 모두 앞뒤가 똑같은 단어를 말함

s 의 길이는 1<= s.length <=16

1. a, a ,b 가 입력시
   - [a,a,b] ,[aa,b] 가 가능함
   - [a,ab] 는 ab 가 팰린드롬이 아니라 불가능
2. 모든 가능한 경우의 수름 탐색해야하므로 깊이 우선탐색을 사용한다 
   - 현재위치(start) 에서 시작해 문자열 한 글짜식 늘려가면 자르나
   - 잘라낸 부분 문자열(substring) 이 팰린드롬인지 확인함
     - 팰린드롬 0 : 리스트에 담고 남은 뒷부분 문자열에대해 재귀적 함수 호출
     - 팰린드로 X : 더 이상 진행하지않고 넘어간다
   -문자열 끝까지 도달시 다음 리스트 정답 목록 반환함
