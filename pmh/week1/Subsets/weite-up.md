# 문제링크

https://leetcode.com/problems/subsets/description/
# 접근방법
1. 숫자가 중복이 안되는 수가 주어지면 그 수를 이용해 가능한 부분집합을 모든 찾는 문제라고생각
2. 가능한 모든 집합을 찾아야되는 완전탐색을 해야겠다고 생각함
3. 재귀 함수를 써서 완전 탐색을 구현해야겠다고 생각함
4. 이때까지 재귀함수를 쓰면 리스트가 주어진 배열과 사이즈가 같아지면 값을 리턴해주었는데 이번문제는 1,2,3 이 넘어오면 1, 2, 3 하나의 값을 리턴해줘야하기때문에 길이가 다른 집합을 어떻게
구할것인지 고민함
5. 재귀 함수를 통해서 탐색하면서 리스트에 쌓이는 과정의 수들이 모드 부분집합이 되므로 저번처럼 배열의 인덱스 와 리스트 사이즈가 같을때 값을 리턴하는게아니라
result 에 값이 추가 될때마다 리턴 해준다
6. 또한 [1.2] [2,1] 은 같은 집합으로 취급되므로 중복이 되지않게 재귀를 호출할때마다 i=1 을 넘겨줘서 현재 선택한 숫자보다 뒤에 있는 숫자들만 들어가게한다
