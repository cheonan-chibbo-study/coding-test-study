# 접근방법
1. 추가하고 싶어하는 코스요리를 구성하는 단품메뉴들의 갯수가 담긴 배열 course 의 수에 맞게 조합을 만들기위해서
    course 에 수에 맞춰 백트래킹을 돌린다 만약 코스가 2,3,4 가 들어온다면 2 일때 요리 갯수 2 가되면 조합을 반환하고 다음 재귀함수를 호출하고
    모든 손님을 통해 가능한 모든 조합 2개를 만들었으면 다음 코스가 3일때 를 다시 반복한다
2. 손님의 주문한 요리의 조합을 만들때마다  주문한 요리 배열을 한번 정렬한다 그 이유는 정렬 안하고 백트래킹시 손님 A 가 ABC 를 손님 B 가 DBA 를 주문시
AB 와 BA 조합이 다른걸로 취급된다.
2. 재귀를 호출할때 List<Map<String,Integer>> list 변수를 선언해  재귀를 list.get(course[n]) 으로 호출해 요리 조합이 2개인에는 리스트의  인덱스 2 에 요리조합이 3개인 애는 리트스의 인덱스 3 에 저장되도록한다
3. 백트래킹 에서 조합이 course 수 랑 똑같을시 map.put(str,map.getOrDefault(str,0)+1) 을 이용해 map에 해당 조합을 넣고 갯수를 더한다
   똑같은 조합이 들어오실 key 의 중복저장안되는 특성을 이용해 key 이 조합은 안바뀌고 값만 +1 이된다
4. 백트래킹을 통해 모든 손님이 주문한 요리 조합 과 그횟수 다 map 을 for(Map<String,Integer> map : list) 를통해 조합이 이 0 개인에 1개인에 2개인에 가져온다
5. if(map.size() >1) 를 통해 손님이 최소 2번이상 주문해야한다는 조건을 만족시킨다
6. max 를 통해 해당갯수 조합중 가장 많이 시켜진 조합을 구한후 결과 리스트에 넣는다
